# 2026-02-04 不明ブロック末尾空行 / ラベル枠線

## 末尾空行の原因
- 不明ブロックは `token.raw` をそのまま保存・表示しているため、
  `:::` ブロックの **終端改行** がそのまま表示に残る。
- Markdown パーサがブロックの末尾に改行を含めるため、
  視覚上 1〜2 行の空白が見える。

## 根本対処の選択肢
- 表示も内容も完全一致を維持するなら空行は仕様。
- 空行を消す場合は tokenizer で末尾改行を除外し、
  表示・保存の「完全一致」を緩める必要がある。

## 変更点
- `.raw-block .block-label` の枠線を削除。
